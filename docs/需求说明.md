# 需求说明

## 背景

我们希望基于 Claude Code 分析 Git 项目，自动识别项目中的创意点、创新方法、源码证据和可迁移价值。

## 最小目标

- 支持输入 Git 仓库地址或本地项目路径。
- 创建异步分析任务。
- 按 6 条扫描线展示处理步骤。
- 调用 `git-creative-discovery` Skill 输出 JSON。
- 保存扫描步骤、创意点和最终报告。
- 提供 Web 页面查看历史任务和报告。

## 暂不实现

- 用户登录和权限系统。
- 多租户。
- 分布式任务队列。
- 自动生成完整 Skill Atom 文件。
- 复杂向量数据库检索。
