# 架构设计

## 总体结构

项目采用前后端分离、部署可单体化的结构。

```text
frontend -> backend -> Claude Code -> Git 项目
                    -> SQLite
```

## 后端职责

- 接收分析任务。
- 克隆远程仓库或读取本地路径。
- 调用 Claude Code Skill。
- 保存任务、步骤、创意点和报告。
- 通过 SSE 推送任务状态。

## 前端职责

- 创建分析任务。
- 展示历史任务。
- 展示 6 条扫描线进度。
- 展示创意点卡片。
- 展示最终报告。

## 数据库

默认使用 SQLite，便于开源用户快速启动。需要正式部署时，可以通过 `DATABASE_URL` 切换到 PostgreSQL。
